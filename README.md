# Mini-Projet SQL — Gestion de Commandes Clients

## Objectif
Créer une base de données PostgreSQL complète pour gérer des clients, des commandes et des produits.

## Compétences acquises
- Modélisation UML (1NF, 2NF, 3NF)
- Création de tables SQL (PK, FK, contraintes)
- Requêtes d’analyse et fonctions de fenêtrage
- Utilisation de PostgreSQL en ligne de commande

## Structure
- create_tables.sql → création de la base
- insert_data.sql → insertion d’exemples
- analysis_queries.sql → requêtes analytiques
- model_UML.png → diagramme UML du projet
